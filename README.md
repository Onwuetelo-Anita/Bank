# Bank Account Management System

## Description

This is a simple Python program that demonstrates the use of **Object-Oriented Programming (OOP)** concepts. The program allows users to create bank accounts, deposit money, withdraw money, and view account balances.

It includes basic banking operations and validation to prevent withdrawals when there are insufficient funds.

## Features

* Create a bank account with an account holder's name.
* Deposit money into the account.
* Withdraw money from the account.
* Display account details and current balance.
* Prevent withdrawals that exceed the available balance.

## Technologies Used

* Python 3
* Object-Oriented Programming (Classes and Objects)

## How It Works

1. A bank account is created with a name and an initial balance of 0.
2. Users can deposit funds into the account.
3. Users can withdraw funds if sufficient balance is available.
4. The program displays the account name and current balance.
